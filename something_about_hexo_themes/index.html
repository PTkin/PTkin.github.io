折腾Hexo的主题

Posted on 2016-05-18

原本写了一篇 《折腾Hexo的搭建与主题》，后来配着配着觉得文章太长，所以把主题推荐搬到这篇博客上了。本文将为大家(我)推荐几个主题，每个主题都给出了截图、Github 链接与 Demo 链接。

关于 Hexo 的搭建与坑，可以点击 这里 查看我的另一篇文章《折腾Hexo的搭建》。

寻找合适的主题

Hexo 是一个博客框架，拥有各式各样丰富多彩的主题可供配置。我们可以几种方式寻找合适的主题：

  1. 通过 官网主题集合 寻找自己满意的主题

  2. 可以直接在 github 上搜索关键词 hexo-theme 寻找

  3. 知乎的提问 有哪些好看的 Hexo 主题？ 也可以作为参考

配置方式各异，主流是在 theme/ 中添加主题的文件夹，然后修改 _config.yml 配置文件。基本上按照 github 上的 readme.md 操作即可。下面几个主题有部分我进行了拓展，只是额外的偏好而已，如果只是按照官方的教程操作的话，很快就能更换好主题。

接下来我将推荐几个我见过的比较好看的主题。

Huxblog

  • GitHub - Kaijun/hexo-theme-huxblog
  • Demo

我在官网的主题中寻找到了一款首页非常棒的主题名 Huxblog，使用效果如下：

桌面浏览器：

贴上使用此主题的优质博客：

  • Kaijun的博客 | Kaijun’s Blog （demo）
  • 黄玄的博客 | Hux Blog （作者）
  • 我思故我在 | ArtinHuang’s Blog

折腾了一会 Huxblog， 觉得首页极美，但代码高亮与html渲染效果不尽如人(我)意，于是又开始捣鼓。

NexT

  • iissnan/hexo-theme-next
  • IIssNan’s Notes

作为 github 上 Hexo 的主题中最热门的一个， NexT 名副其实。使用它还可以换几种内置的主题，支持也很频繁和强大，但是简洁是优点，太过简洁就是缺点。总体上依然好评！

NexT 主题还有官网 : NexT 官网，还有很多优秀的博客例如 Jark’s Blog，github上居然还有个 issue 自发地展示了大家自己的博客，用户量客观 ~

按照 NexT 使用文档 进行配置即可。不得不说 NexT 配置起来比 Huxblog 方便太多 =。=
