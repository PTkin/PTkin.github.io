将Ubuntu安装到移动硬盘

Posted on 2016-05-15

自从手头多余了一个 64G 的固态移动硬盘，一直琢磨着要用它来干嘛。刚好对 Ubuntu 有点感兴趣，也要熟悉熟悉 Linux 操作，那就装个 Ubuntu 来玩玩儿吧 ~

起因

事情的起因是这样的，我给自己 2012 年的 Macbook Air 换上了新的 SSD，淘汰了原有的 64G SSD。于是我拥有了一个 64G 大小的固态移动硬盘，用来做什么好呢？做U盘简直浪费了这固态硬盘，而且我已经有三个正在服役的U盘了，其中2个是 USB3.0 接口，传输速度和容量基本够用。那它还能干什么?

正好最近需要学习 Linux 下的一些操作，要不就给这块移动硬盘装个 Ubuntu？不仅可以熟悉命令还可以玩玩 Ubuntu ～

嗯 想着速度应该没问题，而且在移动硬盘上装系统可能有以下几个好处（未必）：

  1. 随身带着自己的系统和应用程序，在自己熟悉的环境下工作娱乐，不留痕迹
  2. 当电脑有问题的时候，插上别人的计算机可以进行问题检测、资料备份等
  3. （绕过别人计算机的密码进入硬盘 … 且不留痕迹：）

当然，这里可能还会有系统驱动之类的兼容性问题等等，这些问题这里暂时不考虑啦~

安装前准备

由于我们是打算将 Ubuntu 安装到移动硬盘，所以我们需要的有：

  1. 移动硬盘：准备安装操作系统的移动硬盘（或者U盘）
  2. U盘：用来写入操作系统镜像
  3. Ubuntu 镜像：系统镜像（我使用的是 16.04 LTS 版） 官网
  4. UltraISO：镜像写入工具 官网
  5. MiniTool Partition Wizard：分区工具 官网

操作记录

再次说明，此处仅仅是我操作的过程，但是由于笔者本身对 Linux 不是很熟悉，所以安装的过程也多是参考网上的教程，具体何处有什么问题也不甚了解，如果您发现有任何问题，欢迎且十分希望您能指出，谢谢。

对系统动刀前请一定要备份好数据！

一、格式化U盘和移动硬盘

正常操作即可。

二、向U盘写入 Ubuntu 镜像

这一步我使用的是 UltraISO 工具。

使用 UltraISO 写入 Ubuntu 镜像参考：

win7+ubuntu 13.04双系统安装方法_百度经验

其实就是选择 写入硬盘映像，然后选择好 Ubuntu 镜像 和准备写入镜像的 U盘，按以下配置：

  1. 写入方式：USB-ZIP+
  2. 便捷启动：写入新的硬盘主引导记录 -> USB-ZIP+

然后就可以开始写入了，大概几分钟就完成。

三、给移动硬盘分区

原本这一步我打算在安装系统的时候直接使用 Ubuntu 镜像中的安装程序自带的分区工具进行分区，但是一直遇到“无法4K对齐”的问题，所以在这里先进行分区，再安装系统。

这一步我使用的是 MiniTool Partition Wizard 工具。

工具的使用方法教程：

使用MiniTool Partition Wizard对SD卡进行EXT4格式分区_西西软件资讯

分区的配置参考：

win7+ubuntu 13.04双系统安装方法_百度经验

如果你没时间或者没兴趣的话，可以直接按照下面这个配置来分区：

  1. 挂载点：/

    • 大小：20480 MB
    • 类型：主分区
    • 格式：Ext4日志文件系统

  2. Swap 交换空间

    • 大小：2048 MB
    • 类型：逻辑分区

  3. 挂载点：/boot

    （注：也有前辈说这个区不用分，但是身为新手的我还是按照了别人的分区配置。熟悉之后就会了解吧）

    • 大小：256 MB
    • 类型：逻辑分区
    • 格式：Ext4日志文件系统

  4. 挂载点：/home

    • 大小：移动硬盘里剩余所有空间
    • 类型：逻辑分区
    • 格式：Ext4日志文件系统

四、安装 Ubuntu

  1. 在一台计算机上同时插入U盘和移动硬盘

  2. 开机进入 Bios，选择写入了镜像的那个U盘启动

  3. 在选择 安装类型 的时候有3个选项，分别是 安装Ubuntu与原系统共存、格式化磁盘并安装Ubuntu、自定义安装。我们选择 其他选项 进行自定义安装。

  4. 检查自己的移动硬盘的名字与分区是否正确。一般来说 sda 是计算机内置硬盘，sdb 等才是我们插入的外置存储设备。

  5. 安装启动引导器的设备 选择将要安装系统的移动硬盘。

  6. 后面就是一些普通的选项了，随意。

  7. 等待安装。笔者等了一个多小时还未安装完于是睡了一觉第二天起来就ok了：）

Now u can enjoy ur Ubuntu ~!!

接下来笔者要做的

安装完毕后，在计算机的 Bios 中设置从 USB 启动即可选择移动硬盘中的 Ubuntu 启动。

启动正常，使用正常，在安装的本机上没有驱动问题。

操作过程中速度飞快！完全没有受到 USB 连接的限制！比我本子里的 win10 快多了！

接下来要测试能否插在其他机子上进入 Bios 然后从 USB 启动，从而达到最开始的三个目的了！
