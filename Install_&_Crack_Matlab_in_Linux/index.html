Install & Crack Matlab in Linux

Posted on 2016-08-03

Install & crack Matlab in Linux with R2014b_glnxa64.iso, install.jar, license.lic, libmwservices.so.

Install & Crack Matlab (Method 1)

  1. Get MATLAB installation file and CRACK files(includes three files). Here lists what I used:

    • R2014b_glnxa64.iso
    • install.jar
    • license.lic
    • libmwservices.so

  2. Extract R2014b_glnxa64.iso, Here I extract it to [path]/matlab_install

  3. Replace [path]/matlab_install/java/jar/install.jar with install.jar in CRACK

  4. Install Matlab

    While installing, you’ll be asked to enter an product key, but it doesn’t matter what key you enter.(Here I enter 52965-57859-04135-19462, found on the Internet)

  5. Activate Matlab

    • Choose the Activation Method without Internet
    • Then choose the Activation license license.lic in CRACK

  6. Replace /usr/local/MATLAB/R2014b/bin/glnxa64/libmwservices.so with libmwservices.so in CRACK

  7. Open Matlab in Terminal :

    Congratulation!
